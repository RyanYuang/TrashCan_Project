## 📌 快速开始（3步完成集成）

### 第1步：在 main.c 开头添加头文件

```c
/* USER CODE BEGIN Includes */
#include "uart_protocol.h"
#include "protocol_parser.h"
#include "threshold_config.h"
/* USER CODE END Includes */
```

---

### 第2步：在 main() 函数初始化部分添加初始化代码

```c
int main(void)
{
  /* 系统初始化... */
  HAL_Init();
  SystemClock_Config();
  MX_GPIO_Init();
  MX_USART1_Init();  // 确保串口已初始化
  
  /* USER CODE BEGIN 2 */
  
  // 初始化阈值配置（使用默认值）
  ThresholdConfig_Init(NULL);
  
  // 初始化协议解析器
  Protocol_Parser_Init();
  
  // 初始化串口协议（假设使用 USART1）
  UART_Protocol_Init(&huart1);
  
  /* USER CODE END 2 */
  
  while (1)
  {
    // 主循环代码在第3步添加
  }
}
```

---

### 第3步：在 main() 的 while(1) 循环中添加处理代码

```c
while (1)
{
  /* USER CODE BEGIN 3 */
  
  // ===== 1. 检查是否收到完整的指令帧 =====
  if (UART_Protocol_IsFrameReady()) {
      char rxFrame[256];
      UART_Protocol_GetFrame(rxFrame, sizeof(rxFrame));
      
      // 解析协议帧
      Protocol_Parse(rxFrame);
  }
  
  // ===== 2. 读取传感器数据（每500ms发送一次）=====
  static uint32_t lastSendTime = 0;
  if (HAL_GetTick() - lastSendTime >= 500) {
      lastSendTime = HAL_GetTick();
      
      // 这里替换成你的真实传感器读取函数
      float temp = 25.6;    // Read_Temperature();
      float hum = 60.3;     // Read_Humidity();
      float co = 0.5;       // Read_CO();
      float light = 450.0;  // Read_Light();
      
      // 发送数据到上位机
      UART_Protocol_SendSensorData(&huart1, temp, hum, co, light);
  }
  
  /* USER CODE END 3 */
}
```

## 📌 添加中断回调函数

在 `stm32f1xx_it.c` 文件中找到 `HAL_UART_RxCpltCallback` 函数，添加以下代码：

```c
/* USER CODE BEGIN 1 */
#include "uart_protocol.h"
/* USER CODE END 1 */

void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart)
{
  /* USER CODE BEGIN HAL_UART_RxCpltCallback */
  
  // 调用串口协议接收回调
  UART_Protocol_RxCallback(huart);
  
  /* USER CODE END HAL_UART_RxCpltCallback */
}
```

**如果该函数不存在**，在文件末尾添加：

```c
/* USER CODE BEGIN 1 */
#include "uart_protocol.h"

void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart)
{
  UART_Protocol_RxCallback(huart);
}
/* USER CODE END 1 */
```

## 📌 完整示例代码（main.c）

```c
/* Includes */
#include "main.h"
#include "usart.h"
#include "gpio.h"

/* USER CODE BEGIN Includes */
#include "uart_protocol.h"
#include "protocol_parser.h"
#include "threshold_config.h"
/* USER CODE END Includes */

/* USER CODE BEGIN PV */
// 全局变量：当前速度和运动状态
uint8_t g_MotorSpeed = 50;  // 速度百分比
uint8_t g_MotorState = 0;   // 0=停止, 1=前进, 2=后退, 3=左转, 4=右转
/* USER CODE END PV */

/* USER CODE BEGIN PFP */
void Process_ControlCommand(ControlCommand_t cmd);
void Control_Motor(void);
/* USER CODE END PFP */

int main(void)
{
  HAL_Init();
  SystemClock_Config();
  MX_GPIO_Init();
  MX_USART1_Init();

  /* USER CODE BEGIN 2 */
  
  // 初始化模块
  ThresholdConfig_Init(NULL);
  Protocol_Parser_Init();
  UART_Protocol_Init(&huart1);
  
  // 注册指令回调（可选，也可以在主循环中直接获取）
  Protocol_Parser_RegisterCommandCallback(Process_ControlCommand);
  
  /* USER CODE END 2 */

  while (1)
  {
    /* USER CODE BEGIN WHILE */
    
    // 1. 处理接收到的指令
    if (UART_Protocol_IsFrameReady()) {
        char rxFrame[256];
        UART_Protocol_GetFrame(rxFrame, sizeof(rxFrame));
        Protocol_Parse(rxFrame);
    }
    
    // 2. 定时发送传感器数据
    static uint32_t lastSendTime = 0;
    if (HAL_GetTick() - lastSendTime >= 500) {
        lastSendTime = HAL_GetTick();
        
        // 读取传感器（这里用假数据，替换成真实函数）
        float temp = 25.6;
        float hum = 60.3;
        float co = 0.5;
        float light = 450.0;
        
        UART_Protocol_SendSensorData(&huart1, temp, hum, co, light);
    }
    
    // 3. 控制电机
    Control_Motor();
    
    /* USER CODE END WHILE */
  }
}

/* USER CODE BEGIN 4 */

// 控制指令回调函数
void Process_ControlCommand(ControlCommand_t cmd)
{
    switch(cmd) {
        case CMD_STOP:
            g_MotorState = 0;
            break;
        case CMD_FORWARD:
            g_MotorState = 1;
            break;
        case CMD_BACKWARD:
            g_MotorState = 2;
            break;
        case CMD_TURN_LEFT:
            g_MotorState = 3;
            break;
        case CMD_TURN_RIGHT:
            g_MotorState = 4;
            break;
        case CMD_SPEED_25:
            g_MotorSpeed = 25;
            break;
        case CMD_SPEED_50:
            g_MotorSpeed = 50;
            break;
        case CMD_SPEED_75:
            g_MotorSpeed = 75;
            break;
        case CMD_SPEED_100:
            g_MotorSpeed = 100;
            break;
        default:
            break;
    }
}

// 电机控制函数（根据状态控制GPIO/PWM）
void Control_Motor(void)
{
    // 这里添加你的电机控制代码
    // 例如：设置PWM占空比 = g_MotorSpeed
    //      根据g_MotorState控制方向GPIO
    
    switch(g_MotorState) {
        case 0: // 停止
            // HAL_GPIO_WritePin(MOTOR_EN_GPIO_Port, MOTOR_EN_Pin, GPIO_PIN_RESET);
            break;
        case 1: // 前进
            // 设置电机前进方向 + 速度PWM
            break;
        case 2: // 后退
            // 设置电机后退方向 + 速度PWM
            break;
        // ... 其他状态
    }
}

/* USER CODE END 4 */
```

## 📌 测试方法

### 1️⃣ 测试控制指令（从PC发送）

在串口助手或上位机发送以下数据：

- `@1` + 回车换行 → 前进
- `@0` + 回车换行 → 停止
- `@5` + 回车换行 → 速度设为25%

### 2️⃣ 测试阈值配置（从PC发送）

```
#T-10.0,45.0,H20.0,90.0,G30.0,50.0,L0.0,10000.0
```

（后面要加回车换行）

### 3️⃣ 测试数据上报（STM32发送到PC）

每500ms会自动发送一次，格式：
```
@25.6,60.3,0.5,450
```

## 📌 常见问题

### ❓ 编译报错：找不到头文件

**解决**：检查 `.h` 和 `.c` 文件是否都在正确的目录：
- 头文件 → `Core/Inc/`
- 源文件 → `Core/Src/`

### ❓ 收不到数据

**检查清单**：
1. 串口波特率是否为 115200
2. 是否调用了 `UART_Protocol_Init(&huart1)`
3. 是否在中断回调中调用了 `UART_Protocol_RxCallback(huart)`
4. 数据是否以 `\r\n` 结尾

### ❓ 解析失败

检查发送的数据格式：
- 控制指令：`@1\r\n`（不能有空格）
- 阈值配置：`#T-10.0,45.0,H20.0,90.0,G30.0,50.0,L0.0,10000.0\r\n`

## 📌 注意事项

1. **不要在中断中做复杂操作**：中断回调函数只负责接收字节，解析在主循环完成
2. **缓冲区溢出保护**：单帧最大256字节，超出会自动丢弃
3. **如果用了 CubeMX 重新生成代码**：确保 `USER CODE BEGIN` 和 `USER CODE END` 之间的代码不会被覆盖
