# STM32 串口协议使用说明（新手版）

> **正式协议规范**（与当前工程实现逐项对齐）：见同目录下的 [`USART2_通信协议规范.md`](./USART2_通信协议规范.md)。

## 📌 快速开始（3步完成集成）

### 第1步：在 main.c 开头添加头文件

```c
/* USER CODE BEGIN Includes */
#include "uart_protocol.h"
#include "protocol_parser.h"
/* USER CODE END Includes */
```

---

### 第2步：在 main() 函数初始化部分添加初始化代码

```c
int main(void)
{
  /* 系统初始化... */
  HAL_Init();
  SystemClock_Config();
  MX_GPIO_Init();
  MX_USART1_Init();  // 确保串口已初始化
  
  /* USER CODE BEGIN 2 */
  
  // 初始化协议解析器（阈值帧为 #O/G，应用层在 GasCar_App 中注册回调写入 g_EnvCar_Config）
  Protocol_Parser_Init();
  
  // 初始化串口协议（假设使用 USART1）
  UART_Protocol_Init(&huart1);
  
  /* USER CODE END 2 */
  
  while (1)
  {
    // 主循环代码在第3步添加
  }
}
```

---

### 第3步：在 main() 的 while(1) 循环中添加处理代码

```c
while (1)
{
  /* USER CODE BEGIN 3 */
  
  // ===== 1. 检查是否收到完整的指令帧 =====
  if (UART_Protocol_IsFrameReady()) {
      char rxFrame[256];
      UART_Protocol_GetFrame(rxFrame, sizeof(rxFrame));
      
      // 解析协议帧
      Protocol_Parse(rxFrame);
  }
  
  // ===== 2. 周期发送下位机状态帧（示例：每 500ms）=====
  static uint32_t lastSendTime = 0;
  if (HAL_GetTick() - lastSendTime >= 500) {
      lastSendTime = HAL_GetTick();
      UART_StatusUplink_t st = {
          .gas_ppm = 12.5f,
          .obs_flag = UART_OBS_NONE,
          .obs_cm = 0,
          .alarm = UART_ALM_NONE,
          .car_state = 1,
      };
      UART_Protocol_SendStatusFrame(&huart1, &st);
  }
  
  /* USER CODE END 3 */
}
```

## 📌 添加中断回调函数

在 `stm32f1xx_it.c` 文件中找到 `HAL_UART_RxCpltCallback` 函数，添加以下代码：

```c
/* USER CODE BEGIN 1 */
#include "uart_protocol.h"
/* USER CODE END 1 */

void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart)
{
  /* USER CODE BEGIN HAL_UART_RxCpltCallback */
  
  // 调用串口协议接收回调
  UART_Protocol_RxCallback(huart);
  
  /* USER CODE END HAL_UART_RxCpltCallback */
}
```

**如果该函数不存在**，在文件末尾添加：

```c
/* USER CODE BEGIN 1 */
#include "uart_protocol.h"

void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart)
{
  UART_Protocol_RxCallback(huart);
}
/* USER CODE END 1 */
```

## 📌 完整示例代码（main.c）

```c
/* Includes */
#include "main.h"
#include "usart.h"
#include "gpio.h"

/* USER CODE BEGIN Includes */
#include "uart_protocol.h"
#include "protocol_parser.h"
/* USER CODE END Includes */

/* USER CODE BEGIN PV */
// 全局变量：当前速度和运动状态
uint8_t g_MotorSpeed = 50;  // 速度百分比
uint8_t g_MotorState = 0;   // 0=停止, 1=前进, 2=后退, 3=左转, 4=右转
/* USER CODE END PV */

/* USER CODE BEGIN PFP */
void Process_ControlCommand(ControlCommand_t cmd);
void Control_Motor(void);
/* USER CODE END PFP */

int main(void)
{
  HAL_Init();
  SystemClock_Config();
  MX_GPIO_Init();
  MX_USART1_Init();

  /* USER CODE BEGIN 2 */
  
  // 初始化模块（阈值帧见正式规范 #O/G；本模板未演示 RegisterThresholdCallback）
  Protocol_Parser_Init();
  UART_Protocol_Init(&huart1);
  
  // 注册指令回调（可选，也可以在主循环中直接获取）
  Protocol_Parser_RegisterCommandCallback(Process_ControlCommand);
  
  /* USER CODE END 2 */

  while (1)
  {
    /* USER CODE BEGIN WHILE */
    
    // 1. 处理接收到的指令
    if (UART_Protocol_IsFrameReady()) {
        char rxFrame[256];
        UART_Protocol_GetFrame(rxFrame, sizeof(rxFrame));
        Protocol_Parse(rxFrame);
    }
    
    // 2. 定时发送下位机状态帧（正式工程见 GasCar_App 中 EnvCar_StatusUplinkOnce）
    static uint32_t lastSendTime = 0;
    if (HAL_GetTick() - lastSendTime >= 500) {
        lastSendTime = HAL_GetTick();
        UART_StatusUplink_t st = {
            .gas_ppm = 12.5f,
            .obs_flag = UART_OBS_NONE,
            .obs_cm = 0,
            .alarm = UART_ALM_NONE,
            .car_state = 1,
        };
        UART_Protocol_SendStatusFrame(&huart1, &st);
    }
    
    // 3. 控制电机
    Control_Motor();
    
    /* USER CODE END WHILE */
  }
}

/* USER CODE BEGIN 4 */

// 控制指令回调函数
void Process_ControlCommand(ControlCommand_t cmd)
{
    switch(cmd) {
        case CMD_STOP:
            g_MotorState = 0;
            break;
        case CMD_FORWARD:
            g_MotorState = 1;
            break;
        case CMD_BACKWARD:
            g_MotorState = 2;
            break;
        case CMD_TURN_LEFT:
            g_MotorState = 3;
            break;
        case CMD_TURN_RIGHT:
            g_MotorState = 4;
            break;
        case CMD_SPEED_25:
            g_MotorSpeed = 25;
            break;
        case CMD_SPEED_50:
            g_MotorSpeed = 50;
            break;
        case CMD_SPEED_75:
            g_MotorSpeed = 75;
            break;
        case CMD_SPEED_100:
            g_MotorSpeed = 100;
            break;
        case CMD_MODE_MANUAL:
            /* 示例：自行维护模式变量，或调用 EnvCar_Set_Mode(ENVCAR_MODE_MANUAL) */
            break;
        case CMD_MODE_AUTO_TRACK:
            /* 示例：EnvCar_Set_Mode(ENVCAR_MODE_AUTO) */
            break;
        default:
            break;
    }
}

// 电机控制函数（根据状态控制GPIO/PWM）
void Control_Motor(void)
{
    // 这里添加你的电机控制代码
    // 例如：设置PWM占空比 = g_MotorSpeed
    //      根据g_MotorState控制方向GPIO
    
    switch(g_MotorState) {
        case 0: // 停止
            // HAL_GPIO_WritePin(MOTOR_EN_GPIO_Port, MOTOR_EN_Pin, GPIO_PIN_RESET);
            break;
        case 1: // 前进
            // 设置电机前进方向 + 速度PWM
            break;
        case 2: // 后退
            // 设置电机后退方向 + 速度PWM
            break;
        // ... 其他状态
    }
}

/* USER CODE END 4 */
```

## 📌 测试方法

### 1️⃣ 测试控制指令（从PC发送）

在串口助手或上位机发送以下数据：

- `@1` + 回车换行 → 前进
- `@0` + 回车换行 → 停止
- `@5` + 回车换行 → 速度设为25%
- `@9` + 回车换行 → **手动模式**（`CMD_MODE_MANUAL`，正式工程见 `GasCar_App.c` 中 `EnvCar_OnProtocolCommand`）
- `@10` + 回车换行 → **自动循迹模式**（`CMD_MODE_AUTO_TRACK`）

### 2️⃣ 测试阈值配置（从PC发送）

```
#O20,30,G0,500
```

含义：避障触发 20cm、恢复 30cm；气体浓度 0～500 ppm（0 表示关闭下限报警）。行尾须加回车换行。

### 3️⃣ 测试状态帧上报（STM32 → PC）

周期上报（工程中约 300ms），格式见 [`USART2_通信协议规范.md`](./USART2_通信协议规范.md) 第 4.3 节，例如：
```
$STS:12.50,0,0,0,1
```

## 📌 常见问题

### ❓ 编译报错：找不到头文件

**解决**：检查 `.h` 和 `.c` 文件是否都在正确的目录：
- 头文件 → `Core/Inc/`
- 源文件 → `Core/Src/`

### ❓ 收不到数据

**检查清单**：
1. 串口波特率是否为 115200
2. 是否调用了 `UART_Protocol_Init(&huart1)`
3. 是否在中断回调中调用了 `UART_Protocol_RxCallback(huart)`
4. 数据是否以 `\r\n` 结尾

### ❓ 解析失败

检查发送的数据格式：
- 控制指令：`@1\r\n`（不能有空格）
- 阈值配置：`#O<trig_cm>,<safe_cm>,G<low_ppm>,<high_ppm>\r\n`（例：`#O20,30,G0,500\r\n`）

## 📌 注意事项

1. **不要在中断中做复杂操作**：中断回调函数只负责接收字节，解析在主循环完成
2. **缓冲区溢出保护**：单帧最大256字节，超出会自动丢弃
3. **如果用了 CubeMX 重新生成代码**：确保 `USER CODE BEGIN` 和 `USER CODE END` 之间的代码不会被覆盖
